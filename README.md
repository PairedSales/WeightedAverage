# WeightedAverage

Weighted Average is a lightweight browser-based tool for calculating and presenting a weighted average of comparable sales for appraisal reports.

## Current status

This repository now includes the initial Step 1 project skeleton:

- Static HTML, CSS, and JavaScript structure
- Responsive application shell for GitHub Pages deployment
- Modern controls panel and report preview layout
- Placeholder comparable rows ready for calculation logic in the next iteration

## Planned functionality

- Support for 3 to 10 comparable sales
- User-entered weights and adjusted sale prices
- Automatic weighted-average calculation
- Decimal display options for 0, 1, or 2 decimals
- Copy-to-clipboard image export for pasting into appraisal reports

## Development

Open `index.html` directly in a browser to review the current skeleton.
